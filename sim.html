Simplex¶

class simplex.simplexClass¶

Simplex Method and Lemke-Howson to solve zero-sum nash equilibira games

compute_oddments(M, shape_x, shape_y)¶

Oddments method to compute the player strategies if values in matrix are predominantly negative.

Parameters:

  • M – player martix

  • shape_x – matrix x-axis shape

  • shape_y – matrix y-axis shape

Returns:

tuple(List[float], List[float], float): players’ strategies and final game value

compute_saddle_point(M)¶

Computes game saddle point. If game has a pure equilibrium the conditions checked in this method will satisfy that and return the indexes of each players’ pure strategy.

Parameters:

M – player matrix

Returns:

(boolean, List[int, int]): True or False if game has saddle point, indexes of saddle point

create_input_from_matrix(M)¶

Creates a compatible numpy matrix for the simplex method. Used as the matrix returned from the Hotelling method is not suitable

Parameters:

M – Hotelling generated matrix

Returns:

(np.ndarray, np.ndarray): numpy matrix for simplex, numpy matrix to check saddle point

create_input_from_txt_file()¶

Generates a matrix from a text file with a linear program

Returns:

List[List[float]]: matrix of values for simplex program

lemkeHowson(players_matrix)¶

Splits a matrix containing 2 players’ values into two separate player matrices and calls the simplex method to compute the strategies of both players

Parameters:

players_matrix – List[List[List[int, int]]] matrix containing both players’ values

Returns:

tuple(List[float], List[float]): tuple of players strategies

simplex2(matrix)¶

Core simplex method. Calculates the strategies of players utilising tableaus, saddle points and oddments.

Parameters:

matrix – player input matrix

Returns:

tuple(List[float], List[float], float): players’ strategies and final game value
