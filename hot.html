Hotelling¶

class hotelling.HotellingClass¶

Hotelling-Downs model implementation

change_length(length)¶

updates the length of the model line

Parameters:

length – new model line length

change_line(line)¶

updates the values of the line used to calculate player payoff and the line length if greater than initialsied line length

Parameters:

line – List[integer]: list of values

change_pos(pos)¶

updates the candidate positions on the line

Parameters:

pos – List[[int, int]]: list of player positions

change_size(size)¶

Updates the number of strategies for both players

Parameters:

size – number of strategies

create_matrix()¶

creates the payoff matrix for 2 player games (zero-sum) using the line and positions that define the model

Returns:

List[[float, float]]: matrix payoff values for game

create_scenario()¶

creates a random Hotelling scenario (can include 2 ints as parameters deciding number of player strategies)

Returns:

List[[int, int]]: list of players positions

get_length()¶

gets the length of the line

Returns:

int: length of model line
