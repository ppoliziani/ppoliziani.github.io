Hotelling¶

class hotelling.HotellingClass¶

Hotelling-Downs model implementation

change_length(length)¶

updates the length of the model line :param length: new model line length

change_line(line)¶

updates the values of the line used to calculate player payoff :param line: List[integer]: list of values

change_pos(pos)¶

updates the candidate positions on the line :param pos: List[[int, int]]: list of player positions

create_matrix()¶

creates the payoff matrix for 2 player games (zero-sum) using the line and positions that define the model :return: List[[float, float]]: matrix payoff values for game

create_scenario()¶

creates a random Hotelling scenario (can include 2 ints as parameters deciding number of player strategies) :return: List[[int, int]]: list of players positions

get_length()¶

gets the length of the line :return: int: length of model line
