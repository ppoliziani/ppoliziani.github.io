Solver¶

class solver.SolverClass¶
solve_prisoners(p1, p2)¶

Solves n x n pure Nash equilibrium games by maximising or minimising payoffs and finding the cell for which both players play their own strategy

Parameters:

  • p1 – List[]: payoff matrix for player 1

  • p2 – List[]: payoff matrix for player 2

Returns:

List[[int, int]]: list of cells for which a pure Nash equilibrium exists

solve_staghunt(p1, p2)¶

Changes class variables to change type of game from minimising costs to maximising payoff and uses solve_prisoners() method to solve Nash equilibrium

Parameters:

  • p1 – List[]: payoff matrix for player 1

  • p2 – List[]: payoff matrix for player 2

Returns:

List[[int, int]]: list of cells for which a pure Nash equilibrium exists
